## Running
```bash
    > git clone git@github.com:GRAYgoose124/dopamina.git && cd dopamina
    > cargo run
```

## Release build 
```bash
    > cargo build --release --no-default-features
```
Check `target/release/` for your binary named `dopamina`. 
```bash
    > cd target/release/
    > chmod +x dopamina
    
    > ./dopamina
```
